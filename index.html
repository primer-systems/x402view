About x402view

x402view is an investment-focused information source for the x402 ecosystem.

The x402 protocol enables programmatic, gasless payments of any size, including micropayments - turning every API into a monetizable endpoint. As the ecosystem grows, x402view tracks the projects and tokens building in this dynamic new sector.

We focus on investable projects: tokens with real utility, facilitators processing payments, and infrastructure powering agentic commerce.
